Display mourning Black Ribbon at selected corner on every page of your website.

== Description ==

Display mourning Black Ribbon at selected corner on every page of your website. The plugin has schedule capability in which the user can select the period of mourning.

The plugin is created to enable every website to bid farewell to His Majesty King Bhumibol Adulyadej, our beloved king.

Features:

* Selectable placement of the ribbon in four corners
* Schedule-able by start date and end date
* Responsive

Compatibilities:

* All modern browsers including mobiles

== Installation ==

Install using WordPress Plugin Browser or manually place the whole folder into your ‘/wp-content/plugins’ folder.
